# Number Mirror

**Problem Link:** https://www.codechef.com/problems/START01  

Write a program that takes a number **N** as input and prints the **same number** as output.

---

## Input Format

- The input consists of a single integer **N**.

---

## Output Format

- Print the value of **N** in a single line.

---

## Constraints

- `0 ≤ N ≤ 10^5`

---

## Sample Input

123

## Sample Output

123

---

## Explanation

The input number is `123`, so the output is also `123`.

---

## Sample Input

15

## Sample Output

15

---

## More Info

- **Time Limit:** 1 second  
- **Memory Limit:** 1.5 GB  
- **Source Limit:** 50000 Bytes
